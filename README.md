# 个人博客网站搭建教程

欢迎！这个仓库包含了完整的个人博客网站搭建教程和示例代码。

## 🚀 快速开始

**第一次来这里？** 👉 **[快速开始指南](./快速开始.md)** - 告诉你从哪里开始

**想直接看效果？** 👉 进入 `示例代码/my-blog` 文件夹，双击 `index.html`

**想学习搭建？** 👉 **[博客搭建指南](./博客搭建指南.md)** - 完整的入门指南

## 📚 教程目录

1. **[HTML 静态博客教程](./教程/HTML静态博客教程.md)**
   - 从零开始学习 HTML、CSS、JavaScript
   - 适合完全没有编程基础的新手
   - 手把手教你创建第一个博客网站

2. **[Hexo 博客教程](./教程/Hexo博客教程.md)**
   - 使用 Hexo 快速搭建博客
   - 用 Markdown 写文章
   - 更加便捷和强大

3. **[部署指南](./教程/部署指南.md)**
   - 免费部署到 GitHub Pages
   - 让全世界都能访问你的博客
   - 包含详细的步骤说明

## 💻 示例代码

**[示例代码文件夹](./示例代码/)** 包含了一个完整的博客示例：

- ✅ 可以直接使用的 HTML 博客
- ✅ 现代化的设计
- ✅ 响应式布局（支持手机、平板、电脑）
- ✅ 包含首页、文章页、关于页
- ✅ 详细的代码注释

### 如何使用示例代码

1. 进入 `示例代码/my-blog` 文件夹
2. 双击 `index.html` 在浏览器中打开
3. 查看 [示例代码说明](./示例代码/README.md) 了解如何修改

## 📋 学习步骤

1. 阅读 [博客搭建指南](./博客搭建指南.md) 了解全貌
2. 选择一个教程开始学习：
   - 想从基础学起 → [HTML 静态博客教程](./教程/HTML静态博客教程.md)
   - 想快速搭建 → [Hexo 博客教程](./教程/Hexo博客教程.md)
3. 跟着教程一步步操作
4. 查看和修改示例代码
5. 阅读 [部署指南](./教程/部署指南.md) 发布你的博客
6. 开始写作和分享！
